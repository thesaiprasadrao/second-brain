# ⚡ Quick Start (5 minutes)

## 1. Download

```bash
git clone https://github.com/anomalyco/second-brain.git
cd second-brain
npm install
```

**Or** download ZIP from GitHub and extract

## 2. Setup

```bash
npm run setup
```

Fill in:
- Telegram bot token (get from @BotFather)
- Groq API key (free from https://console.groq.com)
- Google permissions (automatic)

## 3. Start

```bash
npm start
```

## 4. Try It!

In Telegram (or terminal):

```
You: check this https://nextjs.org
Bot: Save as "Web Framework: Next.js"? (yes/no)
You: yes
Bot: Saved!

You: what did I save?
Bot: You saved Next.js framework...

You: remind me to code at 5pm
Bot: Reminder set!
```

Done! 🎉
