How It Works

Capture in Natural Language

Send a message to your Telegram bot or WhatsApp. Your brain understands intent:

  • 💡 "cool article about react" → Auto-saved as a capture
  • ✅ "remind me to call mom at 5pm" → Creates a reminder
  • 📝 "add milk to shopping list" → Manages your lists
  • 🔎 "what did I save about next.js?" → Semantic search recall

Example Conversation

You: check this https://nextjs.org
Brain: Save as "Web Frameworks: Next.js"?
You: yes
Brain: ✓ Saved to Google Docs
You: what about react?
Brain: You saved Next.js framework...

Installation

Option 1: One-liner (Recommended)

curl -fsSL https://raw.githubusercontent.com/thesaiprasadrao/second-brain/main/install.sh | bash

Checks Node.js, clones repo, installs deps, creates global command

Option 2: Global npm

npm install -g second-brain

Option 3: Manual

git clone https://github.com/thesaiprasadrao/second-brain.git cd second-brain && npm install

Global Commands

2nd-brain setup

Interactive configuration (API keys, Telegram/WhatsApp)

2nd-brain start

Start daemon (runs in background until you stop it)

2nd-brain status

Check server status, uptime, memory usage

2nd-brain logs [N]

View last N lines of server logs

2nd-brain send "msg"

Send quick message from CLI

2nd-brain stop / restart

Stop or restart the daemon

Quick Start Workflow

# 1. Setup (interactive) 2nd-brain setup # 2. Start daemon (runs forever) 2nd-brain start # 3. Check status 2nd-brain status # 4. Use Telegram to send messages # Server keeps running in background # 5. Stop when done 2nd-brain stop
