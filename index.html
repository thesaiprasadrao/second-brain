Quick Start

Option 1: One-Liner

curl -sSL https://raw.githubusercontent.com/thesaiprasadrao/second-brain/main/install.sh | bash

Option 2: Global NPM

npm install -g @saiprasadrao/2nd-brain

Option 3: Manual

git clone https://github.com/thesaiprasadrao/second-brain.git
cd second-brain && npm install && npm link

Then Setup

1

Run setup wizard

2nd-brain setup

2

Start the daemon

2nd-brain start

3

Send messages on WhatsApp

Watch your brain organize everything
